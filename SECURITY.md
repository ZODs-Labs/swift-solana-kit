# Security

This document specifies the security process for swift-solana-kit.

## Supported Versions

Only the latest minor version of the most recent public major version receives security fixes.

## Reporting

Please report suspected vulnerabilities through GitHub private vulnerability reporting:

https://github.com/ZODs-Labs/swift-solana-kit/security/advisories/new

If GitHub private vulnerability reporting is unavailable, email sol.engineer020@gmail.com with the subject `swift-solana-kit security report`.

Do not file public issues for security reports.

## What To Include

Reports should include:

- The affected version or commit.
- A technical description of the behavior.
- Steps to reproduce the issue.
- Any proof of concept or sample input needed to verify the report.
- Expected impact and any known workaround.

## Response

Maintainers aim to acknowledge valid reports within 7 days. Acknowledgment is not confirmation of the issue. Maintainers may ask for more detail, coordinate a fix privately and publish an advisory after a release is available.
